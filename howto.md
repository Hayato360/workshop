# ทดสอบ Node 

สร้างไฟล์ `index.js` แล้วใส่โค้ดนี้ลงไป

```javascript
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello <<username>> Version 0.0.1');
});

app.listen(port, () => {
    console.log(`Example app listening at http://localhost:${port}`);
});

```


# ทดสอบรันโปรเจค

```commandline
node index.js
```

# สร้างไฟล์ Dockerfile สําหรับ build image

```dockerfile

FROM node

RUN mkdir -p /usr/src/app

WORKDIR /usr/src/app

COPY package.json /usr/src/app/

RUN npm install

COPY . /usr/src/app/

EXPOSE 3000

CMD ["node", "index.js"]

```

# รันคำสั่ง build image

```commandline
docker build -t nodejs-api:uat-v0.0.1 .
```

# รอให้ build image เสร็จ

# รัน container จาก image ที่ build ขึ้นมา

